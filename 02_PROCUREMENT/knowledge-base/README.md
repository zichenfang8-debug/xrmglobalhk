# 02_PROCUREMENT/knowledge-base — 采购知识库

## 用途
沉淀 XRM 在各品类采购中积累的知识：工艺标准、常见问题、供应链洞察、谈判要点。
每一篇知识文章都是可复用的智识资产。

## 分类标签（文件名前缀）
```
[hotel-ffe]   酒店家具固定装置
[hotel-ose]   酒店运营用品
[ai-infra]    AI 基础设施
[electrical]  电气设备
[logistics]   物流与清关
[sourcing]    寻源方法论
[compliance]  合规与认证
```

## 文件命名
```
[分类]-[主题]-v[版本].md
例：hotel-ffe-mattress-sourcing-guide-v1.md
例：logistics-thailand-import-duties-v2.md
```

## 文章模板
```markdown
# [标题]
**品类**: [品类标签]
**更新**: YYYY-MM-DD
**版本**: v1.0

## 核心要点
## 供应商选择标准
## 常见问题与解决方案
## 参考资源
```

## AI 更新规则
- 每完成一个新品类采购任务，生成或更新对应知识文章
- 版本号递增，旧版本不删除（追加 `-archived` 后缀）
