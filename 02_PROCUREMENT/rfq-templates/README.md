# 02_PROCUREMENT/rfq-templates — RFQ 询价模板库

## 用途
存储按品类分类的 RFQ（Request for Quotation）标准模板。
每次发出询价前，从此处选取对应模板，填写项目信息后发送。

## 文件命名
```
RFQ-TEMPLATE-[品类]-v[版本].[docx/xlsx/md]
例：RFQ-TEMPLATE-hotel-ffe-v3.docx
例：RFQ-TEMPLATE-ai-servers-v1.xlsx
```

## 必含字段
- XRM 公司信息（自动填充）
- 询价截止日期
- 产品规格表（含技术参数、数量、交货地）
- 付款条件要求
- 包装与运输要求
- 认证要求
- 报价有效期要求
