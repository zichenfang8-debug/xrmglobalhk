# 02_PROCUREMENT/quotation-templates — 报价模板库

## 用途
存储 XRM 对外发出的报价文件模板：PI（形式发票）、Offer Sheet、报价单。

## 文件命名
```
[文件类型]-TEMPLATE-[用途]-v[版本].[docx/xlsx]
例：PI-TEMPLATE-hotel-ffe-v2.docx
例：OFFER-TEMPLATE-ai-infrastructure-v1.xlsx
例：CI-TEMPLATE-standard-v3.docx
```

## 文件类型说明
| 类型 | 用途 | 必含要素 |
|------|------|----------|
| PI (Proforma Invoice) | 正式报价/预付款依据 | 品名、规格、数量、单价、总价、付款条件、有效期 |
| Offer Sheet | 初步报价/比价 | 品名、规格、价格区间、交期、备注 |
| CI (Commercial Invoice) | 出货后正式发票 | 完整商品信息、HS Code、贸易条款 |

## 保密规定
- 模板中不预填客户名称
- 不预填价格数字
- 不包含 XRM 公司印章

## AI 更新规则
- 每次接到新类型报价需求时，基于最新版本模板创建专属版本
- 版本号递增，保留历史版本
